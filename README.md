## Getting Started

To get a local copy up and running follow these simple example steps.

### Setup

```bash
git clone git@github.com:moise10r/smart_parking_react_dashboard.git
cd ./smart_parking_react_dashboard
```

### Install

```bash
npm install
```

### To Run it

```bash
npm start
```

### To Test it

```bash
npm run test
```


### Build

```bash
npm run build
```
